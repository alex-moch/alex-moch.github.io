Publications.

2023 | Journal of Mathematical Cryptology

Provable security against generic attacks on stream ciphers

Alexander Moch

This paper unifies the earlier SAC 2019 and ToSC 2022 papers. It proves four generic stream cipher constructions to be secure and shows tight bounds with the matching TMDTO attacks.

2022 | IACR Transactions on Symmetric Cryptology

The DRACO Stream Cipher

Matthias Hamann, Alexander Moch, Matthias Krause and Vasily Mikhalev

We improved the model from the earlier SAC 2019 work and proved a new generic stream cipher construction to be secure. Upon this, we proposed a new stream cipher design called DRACO.

2019 | Current Trends in Cryptology

Improving OBDD attacks against stream ciphers

Matthias Hamann, Matthias Krause and Alexander Moch

Ordered binary decision diagrams can be used to obtain a secret internal state from a stream cipher. Our approach lowers the storage demand of such attacks.

2019 | Selected Areas in Cryptography

Tight Security Bounds for Generic Stream Cipher Constructions

Matthias Hamann, Matthias Krause and Alexander Moch

We developed a model for abstracting stream ciphers similar to the Even-Mansour construction and proved tight bounds for two generic stream ciphers schemes.

2019 | Applied Cryptography and Network Security

Parallelizable MACs Based on the Sum of PRPs with Security Beyond the Birthday Bound

Alexander Moch and Eik List

We adapted earlier work on the sum of permutation by Mennink and Neves to a Message Authentication Code and showed security beyond the birthday bound.

2014 | Network and Systems Support for Games

Creation of Custom-made Serious Games with User-generated Learning Content

Philip Mildner, Benjamin John, Alexander Moch and Wolfgang Effelsberg

This paper was written as part of a team project during my Master's. We implemented a serious game in which the player could acquire resources by solving riddles.
